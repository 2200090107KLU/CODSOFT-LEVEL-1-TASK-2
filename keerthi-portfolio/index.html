Projects

Project 1: Online Bookstore Website

This project involved designing and developing a responsive online bookstore website using HTML, CSS, and JavaScript. The website allows users to browse books by category, add them to cart, and proceed to checkout. It also includes a backend implemented with Node.js and Express for managing inventory and processing orders.

Project 2: Student Management System

The Student Management System is a Java-based application developed as part of a semester-long project. It includes features such as student registration, course enrollment, and grade management. The application utilizes MySQL database for storing student information and grades, and provides an intuitive user interface for administrators and students.
